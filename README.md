# ⚽ Sports Analytics Project

A sports analytics project built using Python and SQLite to analyze football match data from the European Soccer Database.

## 📌 Features
- Analyze football match statistics
- SQL queries for data extraction
- Data visualization using Matplotlib
- Top-performing teams analysis
- Home match performance insights

## 🛠 Technologies Used
- Python
- SQLite
- Pandas
- Matplotlib

## 📂 Dataset
European Soccer Database (Kaggle)

## 🚀 Future Improvements
- Interactive dashboards
- Additional visualizations
- Player performance analysis
