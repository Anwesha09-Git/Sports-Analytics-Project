# ⚽ Sports Analytics Project

## 📌 Overview

This project analyzes football match data using Python, SQL, SQLite, Pandas, and Matplotlib.

The data comes from the European Soccer Database and is used to explore team performance, goals scored, wins, and match statistics through SQL queries and visualizations.

---

## 🛠 Technologies Used

- SQL
- Pandas
- Matplotlib

- GitHub

---

## 📊 Visualizations

This project includes:

- Top 10 Teams by Away Matches
- Top 10 Teams by Home Matches
- Top 10 Teams with Most Wins
- Top 10 Teams by Goals Scored
- Top 10 Teams by Goal Difference
- League Average Goals
- Best Home Teams
- Highest Scoring Matches
- Goal Distribution
